Defination List

5 linux distributions which I had tried(chronologically)
Ubuntu
Ubuntu is based on Debian system. Due to this it inherits dpkg. It uses apt package manager. It is one of the most popular OS of linux kernel.
Manjaro
It is a fork of Arch linux. It uses pacman as it's package manager. It supports nvdia drivers out of the box(most of the time). Also, it does almost most of the things what you do in arch linux. Packages are more thoroughly tested before releasing update in compare to arch.
Elementary OS
This OS uses pathogen as it's de which gives feel of mac os. It is also based on debian. So, it's pretty much the same accept of some features.
Pop OS
After Elementary OS, I tried POP os which is also based on debian. So, it's same as Ubuntu. It had 2 iso images one for amd/intel(referring to gc) and other for nvdia. It has it's own nice theme for gnome.
Arch Linux
Minimalistic installation and no bloat as well as bleeding edge distro are some of the specialities of Arch Linux. It has pacman as its package manager, also it has some thirdy party managers like yaourt(discontinued), yayetc. It is rolling release distro.
