Rules Of Game
Welcome to Pig Game

1. Each Player gets to role the dice as many times as he wants and the score is added to roundScore(one in red block).

2. If active player hits 1 all his round score goes back to 0 and now its the chance of next player.

3. Active player can hold his/her round score which means it'll be added to the global score.

4. There are two dices in the game, if both the dices give 6 then the global score of player will become 0.

5. Players can set their high score for which they want to play. They will be provided the option to restart or continue the game if they are in the middle of the game.

6. By default, 200 is set as Top Score.

7. Player reaching the Top Score first will be WINNER.
