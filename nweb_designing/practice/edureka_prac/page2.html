Let's take an example, suppose I opened a text editor in linux system and typed a program to read a file and write something into it. Now, data is stored in _storage drive(HDD, SSD etc)_. So, after `user-process executing` a `get system call` will happen. Like in program `Read()` will happen. Now, as Read() will happen, a **trap()**(interrupt) is set up. Now, I as soon as we shift from user mode to kernel mode, mode bit changes from **1 to 0**. Yes, Mode bit for **User mode** is 1. Mode bit for **Kernel mode** is 0. Now, system call will be executed and after that we'll come back to user mode, because user works in User mode. And, also mode bit changes to 1 from 0.
