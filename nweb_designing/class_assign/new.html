List in HTML tag
A list is a collection of one or more items. There are three types of lists:
  1. Unordered List
  2. Replace List
  3. Definition List

Unordered List

  • An unordered list is created using the <ul> tag.
  • Items in the list are creaed using the <li> tag and displayed in bulllets.


Replace List

  • In this type of list items are displayed using number
  • The list is created using the <ol> tag and items are created using <li> tag


Definition List

  • A definition list is one there list items consist of two parts
    1. Term Part
    2. Definition Description
  • The term part and definition description part of each item created using <dd> and <dt> tag.
